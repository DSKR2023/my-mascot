プライバシーポリシー (Privacy Policy)

デジタルハリウッドサークルDSKR（以下、「当サークル」といいます）は、提供するアプリケーション「Desktop Mascot」（以下、「本アプリ」といいます）における、ユーザーの個人情報およびデータの取り扱いについて、以下の通りプライバシーポリシー（以下、「本ポリシー」といいます）を定めます。

1. Googleデータの利用目的について

本アプリは、ユーザーに以下の機能を提供するためにのみ、Google APIを通じてユーザーデータにアクセスします。

  • Google カレンダー API: デスクトップ上でユーザーの予定を通知・表示するために読み取りアクセスを行います。
  • Google ドライブ API: ユーザーが本アプリ内で作成した設定やメモデータをバックアップ・同期するために、ファイルへの読み書きアクセスを行います。

2. データの保存先と保護

本アプリが取得したGoogleユーザーデータ（認証トークン等を含む）は、すべてユーザーのローカルPC内（端末内）にのみ保存・処理されます。当サークルや開発者を含む外部のサーバーへ送信、収集、蓄積されることは一切ありません。

3. データの第三者提供について

当サークルは、本アプリを通じて取得したユーザーデータを、いかなる第三者にも提供、共有、販売することはありません。

4. Google API Services User Data Policy の遵守

本アプリによるGoogle APIから取得した情報の利用および他のアプリへの転送は、Limited Use要件を含む Google API Services User Data Policy に準拠します。

5. 本ポリシーの変更

当サークルは、法令の変更や機能追加に伴い、必要に応じて本ポリシーを変更することがあります。変更後のプライバシーポリシーは、本ウェブサイトに掲載した時点から効力を生じるものとします。
