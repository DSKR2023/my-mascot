デジタルハリウッドサークルDSKR（以下「当サークル」）は、Desktop Mascot（以下「本アプリ」）および本サイトにおける情報の取り扱いを以下の通り定めます。

1. 取得する情報

本サイトは、問い合わせメールを送信された場合を除き、氏名、住所、電話番号などの個人情報を直接取得しません。 本アプリは、設定やプレイ状態など、アプリの動作に必要な情報をユーザーの端末内に保存する場合があります。

2. Google APIを利用する機能について

Googleカレンダー、Googleドライブ等と連携する機能をユーザーが有効にした場合、本アプリはその機能を提供するために必要な範囲でGoogle APIにアクセスすることがあります。 取得する情報は、予定の表示、設定やメモのバックアップなど、ユーザーが選択した機能の実行目的にのみ使用します。

3. 保存先と管理

本アプリで扱う設定情報、認証に必要な情報、ゲーム進行情報は、原則としてユーザーの端末内で保存・処理します。 当サークルが外部サーバーにユーザーデータを蓄積する機能を追加する場合は、目的、保存内容、保存期間を本ページで明記します。

4. 第三者提供

当サークルは、法令に基づく場合を除き、本アプリを通じて取得したユーザー情報を第三者に販売、共有、提供しません。

5. Google API Services User Data Policy

Google APIから取得した情報の利用および他アプリへの転送は、Limited Use要件を含む Google API Services User Data Policy に準拠します。

6. 広告と外部サービス

本サイトでは、Google AdSense等の広告サービスを利用する場合があります。広告サービスではCookie等が使用されることがあります。 広告を掲載する場合は、本文やアプリ情報などのパブリッシャーコンテンツと関連付く場所に限定し、内容が十分でない画面、エラー画面、操作だけを目的とする画面には広告を配置しません。

8. 改定
